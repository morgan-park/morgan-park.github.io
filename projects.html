Maze Solver Using Reinforcement Learning techniques

(CS138 Reinforcement Learning) Team project

Github

Images of square mazes with different complexities are randomly chosen and downloaded from Google and image processed to transfer to a grid for Markov Decision Process. Different Reinforcement Learning techniques (Dynamic Programming, Temporal Difference, and Deep Q Reinforcement Learning) are used to solve the maze and the performance of the techniques are compared. I led this team project. Technically, I contributed to the project by coding for the image processing, providing the learning environment, and training the model with Dynamic Programming.





Cliff Walking: Comparison of Sarsa and Q-Learning

Github

Cliff walking is an environment of a grid world with the cliff. The task is to find an optimal path from starting state to the goal state without falling off the cliff. I compared the performance of Sarsa and Q-Learning in cliff walking. I also experimented with different hyperparameters.
